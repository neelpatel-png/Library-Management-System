# Library Management System

A Python project for practicing file handling, dictionaries, functions, and menu-driven programming.

## Features

- Add Book
- View All Books
- Search Book
- Borrow Book
- Return Book
- Exit

## Concepts Used

- File Handling
- Dictionaries
- Functions
- While Loop
- Menu-Driven Programming
- Conditional Statements
- String Methods

## Languages Used

- Python 3
